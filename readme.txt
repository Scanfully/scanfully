== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *Scanfully* and click "Install now"
1. Alternatively, download the plugin and upload the contents of `scanfully.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin

= Does the plugin impact my page speed? =
